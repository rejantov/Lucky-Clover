# Lucky Clover

## About This Project

This project was created in **2019**, when I was in **7th grade**. I built everything on my own, without any formal experience, just learning as I went.

Because of that, a lot of the code is hardcoded and not structured the way I would do it today. Still, this project represents one of my first real attempts at building something complete from scratch, and it reflects how I approached problems at the time.

---

## What Is Lucky Clover?

Lucky Clover is a web project focused on **anime and manga content**.

The site includes:

* A homepage where users can browse different series
* A search feature for finding anime or manga
* Pages for episodes and content navigation
* A login system that allows users to keep track of what they like
* A separate section for **manhwa**
* A simple shopping page where users can view items (not functional for purchases)

Even though it is a frontend-only project, I tried to simulate real application behavior. I used **JSON files as a simple database** and stored the data locally.

---

## Technologies Used

* HTML
* CSS
* JavaScript

Everything was written without frameworks or external libraries.

## Project Structure

The project is made up of static pages with styling and JavaScript handling the logic.

Most of the functionality is implemented directly in the frontend, with minimal abstraction.
